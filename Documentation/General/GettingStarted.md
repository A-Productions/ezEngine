Getting Started {#GettingStarted}
===============

IDE to use:
  * Visual Studio 2010 (default)
  * Visual Studio 2008 and 2012 might also work

Build System:
  * For now we generate the solutions with CMake.
  
Build Type:
  * 64 Bit is preferred.
  * 32 Bit should be maintained as well.
