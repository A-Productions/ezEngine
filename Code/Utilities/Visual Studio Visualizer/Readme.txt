This config file enables Visual Studio 2012 to better display the contents of ezString, ezStringBuilder, ezDynamicArray (and other ezArray types) in the debugger.

Simply store this in the following directory:

  "%USERPROFILE%\My Documents\Visual Studio 2012\Visualizers\"
